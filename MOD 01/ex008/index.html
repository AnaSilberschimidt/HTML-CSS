Principais Formatações

Negrito/Destaque

Para uma parte do parágrafo em negrito, usar a tag "b"(sem semântica).

Para destacar uma parte do parágrafo, usar a tag "strong"(com semântica).

Itálico

Nessa frase, temos um termo em itálico usando uma tag "i"(não semântica).

Nessa frase, temos um termo em itálico usando uma tag "em"(semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag "mark".

Texto grande e pequeno

Estamos criando um texto grande com a tag "big" e umtexto pequeno com a tag "small" nesse parágrafo.

OBS: a tag "big" está em vermelho porque é uma tag obsoleta, ou seja, está desatualizada.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag "del".

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois, usando a tag"ins"(semântica).

Existe também o sublinhado com a tag "u"(não semântico)

Texto sobrescristos

Para inserir coisas do tipo x 20+3, usar a tag "sup".

Texto subscrito

Para inserir coisas do tipo H 2 O, usar a tag "sub".
